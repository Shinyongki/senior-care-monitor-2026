# **2025년 노인맞춤돌봄서비스 효과성 검증을 위한 심층 면접(IDI) 가이드**

**조사 개요**

* **목적:** 2차 온라인 설문(정량) 결과의 이면을 파악하고, 구체적인 정책 개선 단서 확보  
* **대상:** 설문 응답자 중 데이터 특이점이 발견된 3개 집단 선별  
* **방법:** 1:1 대면 심층 면접 (약 30\~40분 소요)

## **1\. 성과 우수군 (Best Practice Group)**

**"무엇이 그들을 변화시켰는가?" (성공 요인 모델링)**

* **선별 기준:** 설문 문항 대부분 4.5점 이상 응답, 서비스 이용 후 뚜렷한 긍정적 변화 관찰  
* **면접 목표:** 서비스의 **핵심 성공 요인(Key Success Factor)** 발굴 및 우수 사례 전파

| 번호 | 질문 키워드 | 심층 질문 내용 (Deep-Dive Question) | 질문 의도 (Rationale) |
| :---- | :---- | :---- | :---- |
| **1** | **결정적 순간** | "지난 1년간 서비스를 받으시면서 **'아, 이 서비스 신청하길 참 잘했다'라고 느꼈던 가장 기억에 남는 순간**은 언제였나요?" | **\[성공 경험 포착\]** 추상적인 만족을 넘어, 이용자의 마음을 움직인 결정적 개입(Critical Intervention) 사례 수집 |
| **2** | **핵심 동인** | "건강도 좋아지시고 표정도 밝아지셨는데, **가장 큰 도움이 된 것은 '선생님의 방문(사람)'인가요, 아니면 '프로그램/물품(서비스)'인가요?**" | **\[기여도 분석\]** 인적 요인(라포)과 물적 요인 중 성과에 더 크게 기여한 핵심 동인(Driver) 구분 |
| **3** | **일상의 변화** | "서비스를 받기 전과 비교해서, **아침에 일어나서 주무실 때까지의 하루 일과 중 가장 크게 달라진 점**은 무엇입니까?" | **\[생활 습관 변화\]** 서비스가 노인의 실제 라이프스타일(수면, 식사, 활동)을 어떻게 긍정적으로 재구조화했는지 확인 |
| **4** | **심리적 위안** | "혼자 계실 때 문득 외롭거나 불안할 때가 있으실 텐데, **그럴 때 이 서비스의 어떤 점이 가장 의지가 되시나요?**" | **\[정서적 안전판\]** 고독감 해소에 실질적으로 작동하는 기제(전화, 방문, IoT 등) 파악 |
| **5** | **관계의 확장** | "이제 건강해지셨으니, 혹시 **주변 이웃들에게도 이 서비스를 권하거나, 경로당/복지관에서 활동을 더 많이 하게 되셨나요?**" | **\[선순환 효과\]** 개인의 만족이 지역사회 참여나 선배시민으로서의 역할 확대로 이어지는지 검증 |
| **6** | **신뢰 관계** | "담당 생활지원사 선생님을 가족이나 친구에 비유한다면, **어느 정도나 가깝고 믿음직한 존재라고 느끼시나요?**" | **\[라포의 깊이\]** 서비스 전달 인력과의 신뢰 형성 수준이 성과에 미치는 영향 평가 |
| **7** | **자기 효능감** | "도움을 받는 것을 넘어, 이제는 **'나 혼자서도 이것쯤은 할 수 있다'고 자신감이 생긴 일**이 있으신가요?" | **\[자립 역량\]** 단순 수혜를 넘어 자가 관리 능력(Self-Care)이 얼마나 향상되었는지 확인 |
| **8** | **유지 요인** | "지금 상태를 계속 유지하기 위해서, **앞으로도 이것만큼은 절대로 없어지지 않고 계속 해줬으면 좋겠다** 하는 것은 무엇입니까?" | **\[필수 서비스 식별\]** 예산 감축 시에도 반드시 유지해야 할 최우선 순위 핵심 서비스(Core Service) 파악 |
| **9** | **개선 제안** | "지금도 너무 좋다고 하셨지만, **그래도 욕심을 조금 부려본다면 '이것 하나만 더 있으면 금상첨화겠다'** 싶은 게 있으신가요?" | **\[고도화 단서\]** 우수 이용자 관점에서의 서비스 디테일 개선 아이디어 발굴 |
| **10** | **미래 기대** | "내년에도 이 서비스를 계속 이용하신다면, **본인의 모습이 어떻게 더 좋아지기를 기대하십니까?**" | **\[장기 목표 공유\]** 이용자가 그리는 미래상을 통해 장기적 서비스 목표 재설정 |

## **2\. 데이터 불일치군 (The Paradox Group)**

**"데이터와 현실의 괴리는 어디서 오는가?" (착시 제거 및 진실 규명)**

* **선별 기준:** 객관적 지표(건강/환경)는 나쁜데 만족도는 높거나(착시 가능성), 여건은 좋은데 만족도가 낮은 경우  
* **면접 목표:** \*\*회복 탄력성(Resilience)\*\*의 원인 규명 및 설문 편향(Bias) 제거

| 번호 | 질문 키워드 | 심층 질문 내용 (Deep-Dive Question) | 질문 의도 (Rationale) |
| :---- | :---- | :---- | :---- |
| **1** | **이면의 진실** | "설문조사에서는 매우 만족한다고 하셨는데, **사실 몸은 여전히 많이 편찮으셔서 힘들지는 않으신가요?**" | **\[긍정 편향 검증\]** 서비스 중단을 우려한 방어적 응답인지, 실제 주관적 만족인지 진위 여부 파악 |
| **2** | **만족의 이유** | "(건강이 안 좋음에도 만족한다면) 몸이 힘드신데도 불구하고 **이 서비스를 좋게 평가해주신 가장 큰 이유는 무엇입니까?**" | **\[가치 기준 확인\]** 의료적 호전보다 '정서적 위로'나 '관심' 그 자체에 높은 가치를 두는지 확인 |
| **3** | **숨겨진 조력자** | "우리 선생님 말고도, 혹시 **자녀분이나 종교단체, 이웃 등 따로 도와주는 분**들이 계셔서 생활이 괜찮으신 건가요?" | **\[외부 변수 통제\]** 긍정적 결과가 노인맞춤돌봄서비스만의 효과인지, 가족 등 외부 자원 덕분인지 인과관계 명확화 |
| **4** | **상대적 비교** | "혹시 예전에 **서비스를 전혀 못 받던 시절과 비교해서**, 지금이 훨씬 낫다고 느껴서 점수를 좋게 주신 건가요?" | **\[기준점 확인\]** 현재 상태가 절대적으로 좋은 게 아니라, 과거의 열악함 대비 상대적 만족인지 파악 |
| **5** | **장애 요인** | "열심히 서비스를 받으시는데도 건강이 빨리 좋아지지 않는다면, **무엇이 회복을 가장 방해하고 있다고 생각하십니까?**" | **\[방해 요인 탐색\]** 주거 환경, 경제적 빈곤, 가족 갈등 등 서비스 외적인 장애물 식별 |
| **6** | **심리적 방어** | "혹시 '힘들다'거나 '불만이다'라고 말하면, **선생님이 서운해하거나 서비스가 끊길까 봐 걱정되시나요?**" | **\[사회적 바람직성\]** 솔직한 응답을 가로막는 심리적 장벽(눈치 보기)이 있는지 직접 확인 |
| **7** | **경제적 불안** | "건강 문제보다 당장 **먹고 사는 돈 문제나 난방비 걱정** 때문에 마음의 여유가 없으신 건 아닌가요?" | **\[우선순위 혼선\]** 돌봄 서비스로 해결 불가능한 '빈곤' 문제가 만족도를 왜곡시키는지 확인 |
| **8** | **가족 관계** | "자녀분들과의 관계나 집안 사정이 **서비스 만족도 응답에 영향을 미치기도 했나요?** (예: 자녀가 못 해주는 걸 해줘서 고마움 등)" | **\[가족 대체성\]** 가족 기능의 결핍을 서비스가 대체하고 있는 수준 파악 |
| **9** | **미충족 욕구** | "점수를 후하게 주셨지만, 솔직히 **'나라에서 이것만 해결해주면 소원이 없겠다'** 싶은 딱 한 가지 아쉬움은 무엇입니까?" | **\[사각지대 발굴\]** 높은 만족도 뒤에 가려진, 현재 제도로는 해결되지 않는 구조적 한계 포착 |
| **10** | **설문 이해도** | "지난번 설문조사 때 질문 내용이 어렵거나, **어떻게 대답해야 할지 애매했던 문항**이 있었습니까?" | **\[조사 도구 점검\]** 설문 문항 자체의 난이도나 모호함으로 인한 데이터 오류 가능성 점검 |

## **3\. 특이사례 및 복합 위험군 (Complex Cases & Complaints)**

**"정책의 빈틈은 어디인가?" (사각지대 및 위기 대응)**

* **선별 기준:** 복합 만성질환 \+ 우울 \+ 민원 제기 이력 또는 서비스 거부/중단 위기  
* **면접 목표:** **정책의 빈틈(Gap)** 확인 및 고난도 사례관리 매뉴얼 개선

| 번호 | 질문 키워드 | 심층 질문 내용 (Deep-Dive Question) | 질문 의도 (Rationale) |
| :---- | :---- | :---- | :---- |
| **1** | **불만의 근원** | "지난번에 불편하다고 말씀하셨는데, **구체적으로 어떤 상황에서 '이건 나한테 도움이 안 된다'고 느끼셨나요?**" | **\[실패 지점 확인\]** 막연한 불만이 아닌 구체적인 서비스 실패 지점(Service Failure Point)을 타임라인별로 복기 |
| **2** | **기대 불일치** | "처음에 이 서비스를 신청하실 때 **기대했던 것과, 실제 받아보시니 가장 달랐던 점**은 무엇입니까?" | **\[기대 수준 관리\]** 이용자의 과도한 기대(Expectation)와 실제 제공 가능한 서비스(Reality) 간의 괴리 확인 |
| **3** | **소통 방식** | "담당 선생님의 **말투나 방문하는 태도, 혹은 연락하는 방식**에서 마음이 상하신 적이 있으신가요?" | **\[전달 과정 점검\]** 서비스 내용(Content)이 아닌 전달 태도(Attitude)나 소통 방식의 문제인지 식별 |
| **4** | **자율성 침해** | "혹시 선생님이 건강을 챙겨준답시고 **잔소리를 하거나, 내 생활에 너무 간섭한다고 느끼신 적**이 있습니까?" | **\[자기결정권\]** 돌봄 제공 과정에서 노인의 자율성이나 프라이버시가 침해받았다고 느끼는지 확인 |
| **5** | **서비스 한계** | "어르신의 어려움을 해결하기에 **지금의 방문 시간(주 1\~2회)이 턱없이 부족합니까, 아니면 방문 횟수는 중요하지 않습니까?**" | **\[양 vs 질 식별\]** 물리적인 자원 투입량 부족인지, 서비스 질적 내용의 부적합성인지 구분 |
| **6** | **우선순위** | "몸도 아프고 마음도 힘드신데, **병원 동행, 청소, 말벗 중 당장 하나만 선택해야 한다면** 무엇이 제일 시급하십니까?" | **\[욕구 재조정\]** 복합적인 문제 중 이용자가 체감하는 최우선 순위(Top Priority)를 파악해 개입 순서 재설계 |
| **7** | **낙인감/수치심** | "혹시 생활지원사가 집에 오는 것을 **남들이 알까 봐 꺼려지거나, 도움받는 게 부끄럽다고 느끼신 적**이 있나요?" | **\[심리적 장벽\]** 복지 서비스 이용에 대한 사회적 낙인(Stigma)감이 서비스 거부의 원인인지 파악 |
| **8** | **과거 경험** | "예전에 다른 복지 서비스를 이용하시다가 **안 좋은 기억이나 상처를 받으신 경험**이 있으신가요?" | **\[트라우마 확인\]** 과거의 부정적 경험이 현재 서비스에 대한 불신으로 전이되었는지 확인 |
| **9** | **위기 촉발** | "최근 들어 **가장 화가 나거나, 견딜 수 없이 슬펐던 일**이 있었다면 무엇 때문이었는지 말씀해 주실 수 있나요?" | **\[위기 요인 탐색\]** 서비스 불만의 기저에 깔린 최근의 신변 변화(사별, 사기, 가족 갈등 등) 파악 |
| **10** | **제도 제언** | "우리 복지관 말고 **시청이나 정부에 '노인들을 위해 제발 이런 걸 좀 바꿔달라'고 쓴소리**를 하신다면 무엇입니까?" | **\[정책 제언 근거\]** 현행 지침으로는 해결 불가능한 구조적 문제를 파악하여 상위 기관 건의 자료로 활용 |
